### 五、优化说明（关键调整依据）


1. **排除非游戏域名**：

* 移除通用开放平台：`open.huawei.com`/`open.oppo.com`/`open.vivo.com`/`open.baidu.com`等为全品类开放平台，包含支付、地图等非游戏接口，避免误杀正常服务。

1. **保留核心游戏接口**：

* 新增华为游戏专用接口`gss-cn.game.hicloud.com`：用于游戏玩家信息查询等核心功能，属于纯游戏服务域名。

* 保留厂商游戏 SDK 接口：如`a.ssl.msdk.qq.com`（腾讯游戏 SDK）、`mgbsdk.matrix.netease.com`（网易游戏 SDK），仅作用于游戏场景。



### 六、使用说明

1. **文件路径**：同前（Windows：`C:\Windows\System32\drivers\etc\hosts`；macOS/Linux：`/etc/hosts`）。

2. **生效方法**：修改后执行 DNS 刷新命令（Windows：`ipconfig /flushdns`；macOS：`sudo dscacheutil -flushcache`）。

3. **误杀排查**：若出现非游戏功能异常，可优先检查是否误屏蔽通用 API（本清单已提前规避此类风险）。
