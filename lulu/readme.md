# lulu firewall block list
lulu放回去黑名单 ip / urls

注意IP地址建议只对那些已知的垃圾IP进行屏蔽, 因为一个IP上面可能会有其他的服务域名,屏蔽IP则所有绑定到这个IP的域名都会被屏蔽. 而且失效误伤概率可能会偏高!

所以正常情况下建议使用域名来进行屏蔽.

另外对于chrome浏览器中的网络访问屏蔽,由于系统限制原因,只能使用ip形式来进行屏蔽.
